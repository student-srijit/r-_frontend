# Privacy Policy - Research Plus Extension

Effective date: 2026-04-03

Research Plus Extension is designed to analyze research links using user-provided credentials and account session.

## Data We Process

1. Authentication token for your Research Plus account.
2. Research URL from the active tab or context-menu selection.
3. LLM configuration provided by you (provider, model, API key, optional base URL).
4. Selected research workspace id.

## Storage

1. Extension stores auth token and extension settings in browser extension local storage.
2. API key is used for analysis requests and can be synced from website session by explicit user action.
3. No analytics trackers are included in the extension package.

## Data Sharing

1. Data is sent only to configured backend API endpoint for analysis and account operations.
2. No sale of personal data.

## User Controls

1. You can logout at any time from extension popup.
2. You can clear browser extension storage from browser extension settings.
3. You can disable website session sync by not using the sync action.

## Contact

For support, contact the project owner through the Research Plus website support channel.
